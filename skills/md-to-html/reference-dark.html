Cerkl HTML — Dark Variant Reference

Variant: dark Body: Cosmic base Hero: Cobalt 100 + Cobalt 40 freeform On-brand

This file is the dark-theme overlay for the md-to-html skill. Component class names match the light reference (.hero, .callout, .cmp, etc.) so any artifact type can swap themes without changing markup. One brand-approved gradient lives in the hero block; sections sit as Cosmic 90 panels on a Cosmic base body. Butter is the single warm accent.

When to use the dark variant
High-impact, short-read, external-facing artifacts: leadership briefs, pitch one-pagers, campaign pages, prospect-shareable competitor profiles. Not long-read daily artifacts (recaps, deep-dives) — eye fatigue on 3,000-word reads is real.

01Theme variables

Lift these CSS custom properties as-is to render any artifact type in dark. The gradient hero pattern (--hero-gradient) uses the brand-approved freeform combination #0c1159 + #a7b3ed. One gradient per design — don't stack another elsewhere.
