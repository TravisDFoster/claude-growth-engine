# Insights Video Ideas and Script Outlines


## Short 30s Overview Video Outlines (Product Visuals Only):

### 🎬 Draft 1 — "Feature-Led Walkthrough"

**0–4s — Entry & Orientation**
- Insights dashboard overview visual
- Cursor movement or subtle zoom to establish context

**4–12s — Data Visualization (Primary Focus)**
- Show the main Insights dashboard with charts and graphs loaded
- Scroll or click through visual formats (bar charts, trend lines)
- Pause on a key metric to let the data register

**13–21s — Navigation & Drill-Down**
- Click into a specific report (e.g., email engagement over time)
- Show breadcrumb navigation appearing
- Drill into channel-level or audience-level data

**21–29s — Reporting & Sharing**
- Navigate to a campaign report
- Show dashboard sharing or permission toggle
- Pause on a completed report view so viewers can register the outcome

**29–30s — Wrap-up State**
- Return to Insights dashboard overview


---

### 🎬 Draft 2 — "Task-Based, Single-Flow Narrative"

**0–5s — Starting Point**
- Show Insights dashboard with performance summary tiles
- Subtle zoom to establish context

**5–13s — Exploring Engagement Data**
- Click into email engagement over time report
- Show data loading quickly (performance callout)
- Hover over a data point to reveal detail

**13–18s — Filtering & Custom Ranges**
- Open time range selector
- Select a custom date range
- Dashboard updates to reflect filtered data

**18–24s — Audience-Level Insights**
- Transition to audience or subscriber-level view
- Show individual engagement row or interaction detail
- Hover on a contact record

**24–30s — Final Result**
- Pull back to Insights dashboard overview
- Hold on populated, data-rich view

## Short Individual Separate Feature Animations

Instead of one video, create separate short animations for the 3 key features of Insights.

### Data Visualization
Show the Insights dashboard loading with multiple chart types rendering — bar charts, trend lines, tables. Highlight the transition from one format to another.

### Navigation & Drill-Down
Show a user clicking from the top-level dashboard into a campaign report using breadcrumb navigation. Emphasize the speed and consistency of moving between layers.

### Permissions & Sharing
Animated overlay showing admin vs. team member dashboard views. Highlight the sharing toggle and how access can be scoped to specific dashboards or audiences.
