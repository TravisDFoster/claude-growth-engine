## Notes for downstream owners

- This is a **product launch announcement** (editorial), not an SEO listicle. No keyword brief exists; the keywords above are directional only so the Webflow row isn't empty.
- **Primary Solution = Foundations** rationale: Insights is an analytics feature available to all Broadcast customers, including the free Foundations tier (Foundations accounts have Blasts, Audience Manager, and Insights). So the launch is framed around the value any Broadcast user gets, not enterprise-only capability.
