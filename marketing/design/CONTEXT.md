# Design Context — Cerkl Brand Guidelines

## Brand Rules
- Default to the primary palette (Cobalt 60, Cosmic, Butter); use extended scale colors only when needed
- Use only one gradient per design; choose linear or freeform — not both
- Select the correct logo: product → Broadcast logo, company → Cerkl Wordmark, culture/employees → "How Work Should Be" lockup
- Logo color: color logo on light backgrounds, white logo on dark — never color logo on cobalt or dark backgrounds
- For print: always use CMYK values below — never hex

---

## Gradients

Gradients are a core part of Cerkl Broadcast branding. Use only **one gradient per design**.

**Two types:**
- **Linear** — smooth blend between two colors
- **Freeform** — multi-point custom gradient with unique shapes and color blends

**Approved gradient color range:** `#F5F5FA` → `#A7B3ED` → `#3547C4` → `#4C60D9` → `#0C1159` → `#3547C4`

## CMYK (Print Use Only)

## Logo Usage

### Which logo to use

| Context | Logo |
|---------|------|
| Talking about the **product** | Broadcast powered by Cerkl Logo |
| Talking about the **company** | Cerkl Wordmark |
| Talking about **Cerkl culture or employees** | Cerkl "How Work Should Be" Lockup |

### Logo color rules

| Variant | Use on |
|---------|--------|
| **Color logo** | White or light backgrounds only |
| **White logo** | Black or dark backgrounds only |
| **Black logo** | White or light backgrounds only |

**Never** place a color logo on a cobalt or other dark background — insufficient contrast.  
**Never** use standalone logo elements (e.g. the Broadcast icon without the full wordmark lockup).

Always ensure sufficient contrast between the logo and its background.
