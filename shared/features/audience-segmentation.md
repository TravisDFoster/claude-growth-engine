# Feature: Dynamic Audience Sync & Segmentation

> Part of Cerkl Broadcast. For product overview, see [broadcast.md](../broadcast.md).

**Available on:** All plans (Foundations, Foundations+, Omni AI)

---

## How It Works

Communicators build rules-based audience segments once (by department, location, role, hire date, or any employee attribute) — and Broadcast keeps them current automatically as employee data changes. No IT ticket needed for every send. Manual CSV uploads are also supported for special committees or one-off groups.

## HRIS Integration

Broadcast syncs with every major HRIS system: Workday, SAP, PeopleSoft, ADP, Paycor, and Active Directory. Multiple HRIS sources can be connected simultaneously.

Setup uses a no-code import mapping utility — communicators map incoming HRIS data fields to Broadcast attributes through a guided interface. No custom code, no ongoing IT involvement after setup. The interface shows upload status, record counts, conflict flags, and processing history.

## Segment Permissions

Three approaches to segment access control:

1. **Granular Privacy Controls** — choose between global or restricted access per segment
2. **Decentralized Team Management** — allow individual departments to manage their own subscriber groups
3. **Personalized Dashboards** — segments automatically filter into private, shared, and company-wide views

## Why This Differentiates

Most email marketing tools manage external contact lists that someone maintains manually. Broadcast pulls live org data, so segments reflect the actual org structure at send time — without someone reconciling distribution lists before every campaign.
