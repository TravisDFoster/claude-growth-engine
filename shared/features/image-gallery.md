# Feature: Image Gallery

> Part of Cerkl Broadcast. For product overview, see [broadcast.md](../broadcast.md).

**Available on:** All plans

---

## What It Does

The Image Gallery is a shared library of images your team can reuse across [email-blasts.md](email-blasts.md) and [content-management.md](content-management.md). One upload, available everywhere — reducing redundancy, locking in brand consistency, and saving communicators from re-hunting the same logos and headshots every send.

The gallery is available across all of Broadcast via the Image Gallery icon in the top-right corner. It holds as many images as the organization needs.

## Structure

Each image carries a **Name** (used for search and sort, not visible to readers), **Alt-Text** (ADA-compliant description, also searchable), and **Properties** (file type, dimensions, date uploaded, file size, uploaded by). Browse via search (name or alt-text), sort (date uploaded, alphabetical, last used), and filter by uploader.

**Pro tip:** Reserve specific terms in alt-text — e.g., "Signature" on signature images — to create your own informal search filters.

## Adding Images

Three paths in:

- **Direct upload** — open the gallery, click **Upload Image**, pick a file from your computer.
- **From a Blast** — after uploading to a Blast Image Element, click **Add To Gallery**. Works on existing Blast drafts and templates too.
- **From Content Story Images** — same pattern. Click **Add To Gallery** on any story image, including stories pulled from RSS or other integrations.

Review Name and Alt-Text before saving — defaults pull from the source filename or element fields. Both stay editable later from Image Details. Changing them at point-of-use does not overwrite the gallery copy.

**Accepted file types:** .JPG, .PNG, .GIF. **Max file size:** 1600KB (500KB recommended).

## Image Actions

Hover any image and click the ellipsis (⋮) for **Crop**, **Duplicate**, **View Details**, and **Remove**. Admins get all actions on all images; non-admins can only Crop and Remove images they uploaded themselves. Removing an image does not affect anywhere it is already in use — but the removal itself cannot be undone.

## Image Cropper

Available wherever images live in Broadcast: gallery, Blast Image/Video Elements, Story Images, Content Authors, and Branding Settings. Controls: preset or custom aspect ratio, zoom, 90° rotate, resize and move the crop selection.

## Cropper Errors and Edge Cases

- **Cropping from the gallery is permanent.** There is no undo on a gallery-level crop. To preserve the original, **Duplicate** the image first, or crop the copy inside a Blast or Story instead of in the gallery.
- **Duplicate image names are rejected.** Uploading a file whose name matches an existing Image Name throws an error. Rename the incoming file or rename the existing gallery image to resolve.
